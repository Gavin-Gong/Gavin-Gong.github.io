# line-height 在Android 无法垂直居中, 往上偏移

2018-06-02 更新

字体小于 12px 才会出现这种奇怪的问题

  1. scale, 让字体放大两倍, 然后缩放一半
.content {
    display: inline-block;
    height: 40px;
    line-height: 40px;
    font-size: 20px;
    transform: scale(0.5);
    transform-origin: 0% 0%;
}
  1. table 布局
.container {
  display: table
}
.content {
  font-size: 10px;
  display: table-cell;
  vertical-align: middle
}
  1. 让字体不小于12px 然而这得设计师小姐姐说了算.
